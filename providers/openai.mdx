---
title: OpenAI
---

Koe supports [OpenAI](https://openai.com)'s managed Whisper speech-to-text API.

To use the OpenAI provider you need to have an OpenAI account and an API key. You can sign up for an account [here](https://platform.openai.com/) and create an API key [here](https://platform.openai.com/api-keys).
