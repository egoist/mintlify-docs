---
title: Deepgram
---

[Deepgram](https://deepgram.com/) is a speech-to-text API which offers their own models as well as Whisper models.

They offer $200 credits for everyone so you can use their API with Koe for free.

To use Deepgram, you need to create an account and get an API key. Then enter your API in Koe's Deepgram provider settings.
